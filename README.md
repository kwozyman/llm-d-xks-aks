`llm-d` on `xKS`
===

`llm-d` (https://llm-d.ai/) is a Kubernetes-native distributed inference serving stack helping users serve large generative AI models.

`xKS` in this context refers to major cloud provides Kubernetes services, often abbreviated as `?KS` (for example `AKS` for Azure or `EKS` for AWS).

`llmd-on-xks` provides documentation and deployment scripts for easily running and deploying `llm-d` for those cloud services.

Supported clouds
---

* Azure -- `azure/`

This section to be expanded as cloud provider support is added.

Format
---

For each cloud, there is a subdirectory in this reposity with relevant READMEs and scripting.
